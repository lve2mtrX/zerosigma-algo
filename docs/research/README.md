# Research Reference Library

This directory holds external papers, articles, notes, and extracted hypotheses
for future Claude/Codex review. Saving references here does not train a model and
does not change live strategy behavior.

Every useful reference should be converted into:

1. Summary
2. Testable hypothesis
3. Required data
4. Proposed backtest
5. Status: unreviewed / reviewed / planned / tested / rejected / promoted

Hermes and automated research-library work are deferred until Tasty/RTH quote
readiness and the local paper-test workflow are stable.
