# EOD Review Runbook

Use this after a local paper-test session.

## Checklist

1. Stop paper test if it is still running.
2. Refresh Paper Portfolio.
3. Generate / Refresh EOD summary.
4. Review open and closed trades.
5. Review no-trade reasons.
6. Review quote blockers.
7. Save notes for tomorrow.
8. Compare live paper behavior to backtest expectation.

## What to record

- Profile used.
- Symbol and DTE.
- Whether quotes stayed Available.
- Entry credit and distance to short.
- WDS/corridor context.
- TP/SL/EOD behavior.
- Any mismatch between local paper behavior and historical backtest expectation.

No broker execution or order preview is part of this workflow.
